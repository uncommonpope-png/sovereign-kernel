# brand-guidelines

Apply consistent brand identity rules to all content creation.

## What this skill does
Enforces brand voice, color palette, typography, and tone across documents, emails, social posts, and UI components.

## Brand Profile (Sovereign Kernel / PLT)
- **Primary colors:** Gold (#FFD700 — Profit), Rose (#FF6B9D — Love), Steel (#4A5568 — Tax)
- **Voice:** Direct, sovereign, purposeful. No filler. Every sentence earns its place.
- **Typography:** Monospace for code/data, serif for declarations, sans for action items
- **Tone:** Commanding but warm. Confident, never arrogant.

## Usage Patterns

### Apply brand to email
```
Subject line: short, declarative, no emoji unless intentional
Opening: one sentence, context-setting
Body: bullet points preferred, PLT scoring visible
CTA: single clear action
Sign-off: [Soul Name] | PLT Score: X.XX
```

### Apply brand to social post
```
Format: Hook (1 line) → Insight (2-3 lines) → CTA (1 line)
Hashtags: #PLT #SovereignKernel #AI max 3
Tone: Visionary, not salesy
```

### Apply brand to document
```
Title: ALL CAPS or Title Case
Section headers: ## bold, PLT-annotated
Data tables: include P/L/T columns where applicable
Footer: "Built by Craig Jones — Grand Code Pope | Woven by Seshat"
```

## Example commands
```
ACTION: Apply brand guidelines to the draft announcement text
ACTION: Rewrite this email in sovereign kernel brand voice
```
